# Screen Monitor PiP Walkthrough

โปรแกรมนี้จำลองฟีเจอร์ "Picture-in-Picture" (PiP) สำหรับหน้าจอคอมพิวเตอร์ของคุณ โดยให้คุณเลือกพื้นที่ใดก็ได้บนหน้าจอและนำมาแสดงผลในหน้าต่างย่อส่วนที่อยู่บนสุดเสมอ

## วิธีการใช้งาน

1. **การเริ่มต้น**: เมื่อรันโปรแกรม หน้าต่าง "Select Window to Monitor" จะปรากฏขึ้น
2. **การเลือกแหล่งที่มา**:
    - **เลือกโปรแกรม**: เลือกรายชื่อโปรแกรมจาก Dropdown แล้วกด "Start Capture" เพื่อแคปหน้าต่างนั้น (โหมดนี้รองรับการแคปหน้าต่างที่โดนโปรแกรมอื่นบังอยู่)
    - **เลือกพื้นที่แมนนวล**: กด "Select Area Manually" เพื่อลากพื้นที่ครอบเอง (โหมดนี้จะเห็นสิ่งที่อยู่บนหน้าจอจริงๆ เท่านั้น หากโดนบังจะเห็นสิ่งที่มาบัง)

> [!IMPORTANT]
> **การแคปหน้าต่างเบื้องหลัง (Background Capture):**
> - โหมดเลือกโปรแกรมจะใช้เทคนิค `PrintWindow` ซึ่งทำให้คุณเห็นภาพโปรแกรมเป้าหมายได้แม้จะมีหน้าต่างอื่นมาบัง
> - หากหน้าต่างเป้าหมายถูก **พับ (Minimized)** ลง Windows อาจจะหยุดส่งข้อมูลภาพ ทำให้ภาพนิ่งหรือเป็นสีดำในบางกรณีครับ
> - สำหรับโปรแกรมที่ใช้การเร่งด้วยฮาร์ดแวร์ (เช่น Chrome หรือเกมบางเกม) หากแคปแล้วเป็นสีดำ แนะนำให้ลองปิด "Hardware Acceleration" ในการตั้งค่าของโปรแกรมนั่นๆ ครับ
3. **การควบคุมหน้าต่าง PiP**:
    - **ลาก (Drag)**: คลิกซ้ายค้างที่หน้าต่างเพื่อลากไปวางที่ใดก็ได้บนหน้าจอ
    - **ย่อขยาย (Resize)**: ลากที่มุมขวาล่างของหน้าต่างเพื่อปรับขนาด
    - **เมนูคลิกขวา**: คลิกขวาบนหน้าต่างเพื่อเลือก "Reset Selection" (เลือกพื้นที่ใหม่) หรือ "Exit" (ปิดโปรแกรม)
4. **ปุ่มลัด (Hotkeys)**:
    - `R`: ล้างพื้นที่เดิมและเลือกพื้นที่ใหม่ (ในหน้าต่าง PiP)
    - `Esc`: ปิดหน้าต่างหรือยกเลิกการเลือก

## การรันโปรแกรม

1. **ใช้งานไฟล์ EXE (แนะนำ)**:
   - เข้าไปที่โฟลเดอร์ `dist`
   - รันไฟล์ `CaptureMonitorPiP.exe` ได้ทันทีโดยไม่ต้องติดตั้ง Python
2. **รันบน Terminal**:
   - เปิด Terminal ในโฟลเดอร์โครงการแล้วรัน:
   ```bash
   python main.py
   ```
